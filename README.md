# Ting Liang Personal Website

This repository hosts the personal academic website of Ting Liang.

The website includes:

- biography and contact information
- research interests and news updates
- representative and full publication lists
- software/projects (e.g., `pySED`)
- downloadable CV and external links (Google Scholar, GitHub, ORCID)

The site is served via GitHub Pages and will be updated with new research outputs and project information.
